# Select

Custom HTML select component with Vue rendering and binding.

## Prerequisites

- Vue.js 3
- Tailwind CSS

## Examples

<select>
    <option>Test</option>
</select>

### Usage
``` js-vue
<select>
    <option>Test</option>
</select>
```
